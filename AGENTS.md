# Ma Creatives Studio Website Blueprint

This is the standard operating system for all websites built by Ma Creatives Studio. Every site generated must adhere to these professional, creative, and market-ready standards.

## Brand Personality
Modern, Bold, Creative, Professional, Trustworthy.

## Design Philosophy
- Make hard things simple.
- Make simple things beautiful.
- Make beautiful things unforgettable.

## Core Visual Signature
- Modern fonts (Inter for sans, Outfit/Space Grotesk for display).
- Clean spacing & Rounded corners.
- Glassmorphism & Gradient accents.
- Premium animations (motion/react).

## Mandatory Signature Footer
Every website MUST include the developer signature:
- **Slogan**: "Need a website like this? Talk to Ma Creatives" (or approved alternatives).
- **Link**: `https://macreativesstudio.github.io`
- **Behavior**: Opens in a new tab, bold/attention-grabbing color, hover animation.

## Technical Standards
- Mobile-first responsive design.
- Performance optimized (fast loading).
- SEO-ready structure.
- Clean, organized TypeScript code.
- Functional navigation and contact systems.

## Universal Website Structure
1. **Home**: Hero (Big headline + CTA), About Preview, Services Preview, Featured Work, Testimonials, CTA.
2. **About**: Story, Mission/Vision, Team, Timeline.
3. **Services**: Service cards with icons, descriptions, and CTA.
4. **Portfolio/Store**: Filterable/Searchable asset cards.
5. **Contact**: Validation-ready form, WhatsApp integration, Social links.
